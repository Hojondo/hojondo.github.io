# 小米 4 刷机

_最近因为旧的安卓备用机硬件瓦特了，现在二手不到 200，维修费用 70，算了，又入手一个刷机神器-小米 4，几年不碰刷机了，这次摸索花了一天时间，折腾！_

## 一、刷入 recovery(TWRP)

- [官网](https://twrp.me/Devices/Xiaomi/)找到自己的设备

  坑：小米 4 代号是[cancro](https://www.miui.com/thread-2628980-1-1.html)，同米 3

- 下载[ADB(Fastboot)工具](https://www.moonlol.com/adb-fastboot%E5%B7%A5%E5%85%B7%E7%B7%9A%E5%88%B7twrp-recovery-5954.html)，把下载好的 twrp.img 放在该压缩包文件夹[adb](https://pan.baidu.com/s/11ATtwFH8J7o7SApJCPnOIQ)内

- 手机进入 Fastboot 模式，数据线连接 Windows

- 在**adb**文件夹内运行 cmd.exe，依次输入以下[命令](https://zhuanlan.zhihu.com/p/34412300)：

  > ```shell
  > adb devices
  > adb reboot bootloader
  > fastboot flash recovery twrp.img
  > fastboot boot twrp.img # 此行命令可以进TWRP的rec（或者刷入TWRP后重启前按音量加减和电源键进手机rec模式）
  > fastboot reboot
  > ```

- 重启，刷入成功。`电源键`+`音量+`进入 recovery。

## 二、刷入 ROM

- 去 miui[波兰版官网](https://miuipolska.pl/download/)，找到自己的设备**_Mi 4 W/LTE_**，下载 ROM zip 包
- 手机进入 TWRP，先`format data`或`advanced wipe` [四清](https://blog.csdn.net/DXCyber409/article/details/80674946)操作
- `Reboot system`重启选择**重启**到 Recovey（[重要！](https://www.miui.com/forum.php?mod=viewthread&tid=6234383&page=1#pid160860357)）
- 手机保持开着 TWRP 模式，数据线连接电脑，文件传输复制 zip 包到手机存储空间，TWRP 点击`install`安装该 ROM
- 安装完成自动重启进入系统。

## 三、获取 ROOT

目前 root 不再是 superSU 的代名词(附上[superSU 选择](https://prototypez.github.io/2016/05/16/root-and-install-Xposed-framework-on-XiaoMi4-with-Android-M/))，随着 suerSU 原作者 chainfire 不再更新，已经不能实现一加氢氧系统安卓 8.0 和更高版本的 ROOT 了，越来越多的人转向 magisk。[教程来源](https://www.oneplusbbs.com/thread-4077732-1.html)

刷入[Magisk](https://sspai.com/post/53809)框架之后，默认就会具备 ROOT 权限获取和管理啦，而且在 Magisk 的管理器里，还有针对指定 app 隐藏 root 权限检测的 MagiskHide，以免某些软件发现 root 了就直接自我崩溃，不给使用的情况。

- 下载[magisk](https://www.xda-developers.com/how-to-install-magisk/) zip 包
- TWRP`install`安装

## 四、下载太极模块或 xposed 模块

[注意：太极不支持 android7.0 以下系统](https://github.com/taichi-framework/TaiChi/issues/606)

所以暂时选择[Xposed magisk 版](https://www.oneplusbbs.com/thread-4037966-1.html)

- 下载 Xposed Installer-magisk app（magisk 模块详情中的地址已失效，在[androidfilehost](https://androidfilehost.com/?fid=5862345805528068090)下载），在 app 内在线安装 zip 包
- 进入 magisk 模块下载 menu，搜索`Xposed framework`，下载安装 zip 包
- 重启完成

## 五、上梯子

下载[Go Hosts](https://github.com/Lerist/Go-Hosts) root 改 host 文件代替 VPN，专用源需要付费，其他源全不可用。
